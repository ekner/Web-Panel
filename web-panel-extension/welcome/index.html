Web Panel

First time startup


  • Click the plus button to add a bookmark.
  • Click the bookmarks button (to the right) to view your bookmarks. Right-click to delete a bookmark.
  • Click the expand button (three dots) to display the back and forward buttons.
  • Right click the expand button to toggle between mobile mode (green) and desktop mode (gray). Mobile mode will display the mobile version on some sites.
  • Right click the reload button to display options for auto refresh.
  • In the bottom bar you can:
    • Switch between dark and light theme.
    • Zoom on the page.
    • Open the current page in a browser tab.
  • If you want to change other settings, you can go to the options page. Do that by first navigating to the extensions page in your browser, locate Web Panel, and then click options.
